### Annotation Processor

There is also a helpful annotation processor for generating component builders, as writing correct JSignal builders by hand is fairly tedious.

<Tabs syncKey="buildSystem">
  <TabItem label="Maven">
    For maven usage with
    [`org.apache.maven.plugins.maven-compiler-plugin`](https://maven.apache.org/plugins/maven-compiler-plugin/).
    <Code
      code={`<configuration>
    <annotationProcessorPaths>
        <annotationProcessorPath>
            <groupId>org.jsignal</groupId>
            <artifactId>prop</artifactId>
            <version>${version}</version>
        </annotationProcessorPath>
    </annotationProcessorPaths>
</configuration>`}
      lang="xml"
    />
  </TabItem>
  <TabItem label="Gradle (Kotlin)">
    <Code
      code={`annotationProcessor("org.jsignal:prop:${version}")
compileOnly("org.jsignal:prop:${version}")`}
      lang="kotlin"
    />
  </TabItem>
  <TabItem label="Gradle (Groovy)">
    <Code
      code={`annotationProcessor 'org.jsignal:prop:${version}'
compileOnly 'org.jsignal:prop:${version}'`}
      lang="groovy"
    />
  </TabItem>
</Tabs>
